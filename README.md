# Aussie English Daily Quest

An offline-first vocabulary and daily task game for practising practical Australian English.

## What is included

- Daily vocabulary quests with three focused words per day
- Meaning-match mini game
- Real-life speaking and writing prompts
- Section-level input controls for each learning area
- End-user walkthrough steps
- Local progress, points, and levels with `localStorage`
- PWA manifest and service worker for offline use
- Lightweight UI with no remote images, fonts, or APIs
